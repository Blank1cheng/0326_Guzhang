# 故障注入平台

当前仓库保存的是静态可运行版本，主入口为 [index.html](./index.html)。

## 目录

- `index.html`: GitHub Pages 默认入口
- `TEST.html`: 当前主开发版本
- `Sample.html`: 参考页面
- `PROCESS_LOG.md`: 过程记录
- `assets/`: 早期拆分版样式和脚本资源

## 本地运行

直接用浏览器打开 `index.html` 或 `TEST.html` 即可。

## GitHub Pages

推送后可通过 GitHub Pages 发布静态站点，预期访问地址：

`https://blank1cheng.github.io/0326_Guzhang/`

如果仓库的 Pages 还未开启，请在仓库 `Settings > Pages` 中将来源切换为 `GitHub Actions`。
